# ProductSubProduct
ProductSubProduct with Springboot CRUD
Created as Part of an Interview Round within the time span of 2 hours
Main Objectives:-
 Create ,Read ,Update ,Delete products 
 Use of various Annotations such as @autowired @restcontroller @service @entity @data @lombok @Data @setter @getter @requestmapping @postmapping @ putmapping @getmapping @deletemapping
Implementation of swagger/postman to call these restapi's to check that the code functions appropriately
